=========================
Windows 8.1
=========================


.. seealso:: http://fr.wikipedia.org/wiki/Windows_8.1


Windows 8.1 est une mise à jour gratuite de Windows 8 qui fera son apparition 
durant l'année 2013, comprenant notamment une mise à jour d'Internet Explorer.
