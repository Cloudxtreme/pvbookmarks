What is teamviewer ?
=====================

Avec TeamViewer, vous pouvez vous connecter à n’importe quel ordinateur dans le
monde en quelques clics seulement.

Vous prendrez ainsi la main sur le PC de votre partenaire comme si vous y étiez !
