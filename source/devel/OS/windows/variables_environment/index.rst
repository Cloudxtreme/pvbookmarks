La commande ``set`` pour Initialiser une variable d'environnemnt
=================================================================

.. seealso::

   - http://ss64.com/nt/set.html

Sous Windows, la commande set permet de créer, de modifier et d'afficher les 
variables d'environnement. 

Néanmoins, les modifications interviennent uniquement dans la session en cours.

Pour afficher une variable, la commande est la suivante::

    set NomVariable

Pour créer une variable, la commande est la suivante::

    set NomVariable=valeur

Pour supprimer une variable, la commande est la suivante::

    set NomVariable=
