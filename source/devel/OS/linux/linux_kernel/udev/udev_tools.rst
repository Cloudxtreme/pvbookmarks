udevdiscover

.. seealso:: http://fontanon.org/udevdiscover/download-and-install/

Installation tips
=================

Udev-discover GUI has been developed with Gtk3. It needs Gobject Introspection
Repository (gir) bindings for python.

Dependencies:

- python >= 2.6
- python-gobject
- python-gudev
- gir1.2-gtk-3.0
- gir1.2-gconf-2.0
- gir1.2-gdkpixbuf-2.0
