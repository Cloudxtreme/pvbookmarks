Introduction
============

POSIX embedded realtime operating system.

- POSIX 1003.1 api and realtime extension. mixed PSE53/PSE52 profile 
  (see POSIX 1003.13)
- Realtime scheduler (Realtime kernel abstraction layer support eCos 3.0 
  and embOS) .
- deeply embedded platform (ARM7/ARM9, cortex M3/M4, Linux/win32 simulation).
- device driver interface.
- vfs virtual file system (support fat, yaffs and other specific file system).
  streams like system. 

You can request help and ask questions on lepton-discuss@googlegroups.com.

You can ask your questions at #lepton on chat.freenode.net.
