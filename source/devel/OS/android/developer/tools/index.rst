Introduction
============

.. seealso::

   - https://developer.android.com/sdk/index.html
   - :ref:`adt_eclipse_plugin`


The ``Android Developer Tools`` (**ADT**) plugin for Eclipse provides a professional-grade
development environment for building Android apps.

It's a full Java IDE with advanced features to help you build, test, debug, and
package your Android apps.

Free, open-source, and runs on most major OS platforms.
To get started, download the `Android SDK`_.


.. _`Android SDK`:  https://developer.android.com/sdk/index.html
