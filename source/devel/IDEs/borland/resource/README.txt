===============
README.txt file
===============

The 'firmware.rc' file is used by 2 GUI borland projects:

- XLOG8X235_UpdateReaderFirmwareLight
- XLOG8X236_UpdateAllReadersFirmwareLight

This file contains the last firmwares for the CL1356T & T5 readers.


Update rules
============

When a new fimrware is available, follow these instructions:

- update the Firmware.rc file
- compile the file in the Borland IDE
