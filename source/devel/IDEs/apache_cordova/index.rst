Introduction
=============

C'est un framework qui permet de développer des applications en HTML, CSS 
et Javascript et de les transformer en applications natives pour:

- Android 
- iOS
- Blackberry
- etc...

L'intérêt est de développer le code une fois et le distribuer sur l'ensemble des
plateformes mobiles.

======
