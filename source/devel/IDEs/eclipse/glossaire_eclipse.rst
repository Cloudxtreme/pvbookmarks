.. glossary::


    Workspace
    
        Le workspace est le répertoire dans lequel sont stockés :
        
        - les projets, c'est-à-dire les différents programmes que vous allez réaliser
        - la configuration d'eclipse

        Afin d'éviter de reconfigurer eclipse plusieurs fois, on évite en général 
        de multiplier les workspaces.

        .. seealso:: http://www.enseignement.polytechnique.fr/informatique/profs/Julien.Cervelle/eclipse/#workspaceTitle
