Description
===========

``WordPress`` est un système de gestion de contenu libre écrit en PHP, 
reposant sur une base de données :ref:`MySQL <mysql>` , et distribué par Automattic. 

``WordPress`` est surtout utilisé comme moteur de blog, mais ses fonctionnalités 
lui permettent également de gérer n'importe quel site web. 

Il est distribué selon les termes de la GNU GPL. 

Le logiciel est aussi à l'origine du service WordPress.com.
