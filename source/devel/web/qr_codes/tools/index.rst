Python QR Code image generator
===============================

.. seealso::

   - https://pypi.python.org/pypi/qrcode


This module uses image libraries, Python Imaging Library (PIL) by default, to 
generate QR Codes.

It is recommended to use the pillow fork rather than PIL itself.
