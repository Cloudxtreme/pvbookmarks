Description
===========

Internet Explorer (officiellement Windows Internet Explorer depuis la 
version 7, anciennement Microsoft Internet Explorer), parfois abrégé 
IE, MIE ou MSIE, est le navigateur Web développé par Microsoft, installé 
par défaut avec Windows. 

Depuis qu'il a détrôné Netscape Navigator à la fin des années 1990 et 
jusque vers 2012, c'était le navigateur Web qui a le plus été utilisé 
au monde. 

Pourtant, en 2014, il n'est plus que deuxième en Europe et troisième 
en France. 

Ses principaux concurrents sont Mozilla Firefox (depuis 2004) et Google 
Chrome (depuis 2008) et Safari.

Les versions antérieures à la version 7 sont nommées 
**Microsoft Internet Explorer**, tandis qu'à partir de la version 7, le 
logiciel est intitulé **Windows Internet Explorer**.
