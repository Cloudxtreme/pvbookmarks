.. _feedly:

=======================
Feedly extension
=======================

.. seealso::

   - http://aldarone.fr/bye-bye-google-reader-bonjour-feedly/

L’extension ajoute également un petit bouton à moitié transparent nommé
Feedly-mini en bas à droite de chaque site (il est évidemment possible de
le désactiver dans les options.) Il permet de s’abonner rapidement aux flux
du site consulté ou encore de partager la page sur twitter, facebok, e-mail,
etc..
