Git for pitivi
==============

.. seealso::

    - http://wiki.pitivi.org/wiki/Git

This page is not meant to be a general tutorial for Git; for that, see the
`GNOME Git page`_, the `official Git tutorial/documentation`_ page and `git ready`_.

In this page, we will cover some more advanced usage and the specifics of how
we use Git in the Pitivi project.

This is aimed at people coming from Bazaar or Subversion.
