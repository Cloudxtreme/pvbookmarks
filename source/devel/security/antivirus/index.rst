===================
Antivirus software
===================


.. seealso::

   - http://en.wikipedia.org/wiki/Antivirus_software


Antivirus or anti-virus software is used to prevent, detect, and remove malware,
including but not limited to computer viruses, computer worm, trojan horses,
spyware and adware.

This page talks about the software used for the prevention and removal of such
threats, rather than computer security implemented by software methods.
