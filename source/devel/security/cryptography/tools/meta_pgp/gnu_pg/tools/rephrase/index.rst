========================================
Rephrase
========================================


.. seealso::

   - http://www.roguedaemon.net/rephrase/


``Rephrase`` is a specialized passphrase recovery tool for GnuPG.  

If you can nearly remember your GnuPG passphrase - but not quite - then Rephrase 
may be able to help.
