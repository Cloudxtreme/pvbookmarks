Introduction
============

Il existe plusieurs serveurs de clés usuels. 

Citons par exemple les serveurs 

- pgp.mit.edu, 
- www.keyserver.net, 
- wwwkeys.pgp.net, 
- wwwkeys.us.pgp.net, 
- wwwkeys.uk.pgp.net, 
- etc. 



La commande::

    host -l pgp.net | grep wwwkeys 
    

donne une liste non exhaustive de serveurs de clés.
