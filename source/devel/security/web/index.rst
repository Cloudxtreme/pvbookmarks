The Web Application Security Consortium (WASC)
==============================================

The Web Application Security Consortium (WASC) is 501c3 non profit made up of
an international group of experts, industry practitioners, and organizational
representatives who produce open source and widely agreed upon best-practice
security standards for the World Wide Web.
