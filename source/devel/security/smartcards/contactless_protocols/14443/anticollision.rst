===========================
ISO/IEC 14443 anticollision
===========================

Radio interface - Type A
=========================

PCD to PICC
-----------

Carrier: 13.56Mhz
Modulation ASK 100%
Coding: Modifed Miller
Bandwidth: 106kbit/s  -> 847kbit/s

PICC to PCD
-----------

Load modulation
Subcarrier: fc=16 -> fc=128
Coding: OOK, Manchester
Bandwidth: 106kbit/s -> 847kbit/s

Radio interface - Type B
========================

PCD->PICC
---------

Carrier: 13.56Mhz
Modulation: ASK 10%
Coding: NRZ
Bandwidth: 106kbit/s->847kbit/s


PICC->PCD
---------

Load modulation
Subcarrier: fc=16 -> fc=128
Coding: BPSK, NRZ-L
Bandwidth: 106kbit/s->847kbit/s
