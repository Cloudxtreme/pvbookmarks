===============
flash language
===============

.. seealso::

   - http://googlecode.blogspot.com/2011/11/flash-developers-export-to-html5-with.html
   - http://www.lemondeinformatique.fr/actualites/lire-adobe-donne-flex-a-la-fondation-apache-46682.html



Même si l'éditeur assure qu'il va continuer à supporter le framework, ainsi que
la technologie sous-jacente Flash, il reconnaît aussi qu'à l'avenir les
développeurs d'applications web vont plutôt utiliser HTML5 que Flash.

**A long terme, nous croyons que HTML5 sera la meilleure technologie pour
concevoir des applications d'entreprise**, a indiqué la société dans un
récent billet de blog.

Nous savons aussi qu'actuellement, Flex apporte des bénéfices sur les projets
à grande échelle, typiquement ceux qui comportent une déclinaison sur les postes
de travail.
