Introduction
============

La Low Level Virtual Machine (LLVM, en français : « machine virtuelle de bas
niveau ») est une infrastructure de compilateur conçue pour optimiser la
compilation, l'édition de liens, l'exécution et les temps morts dans un
programme écrit dans un langage quelconque.
