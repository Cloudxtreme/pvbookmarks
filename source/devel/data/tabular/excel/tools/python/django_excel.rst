===================
Sphinx Exceltable
===================

.. seealso::

   - http://packages.python.org/sphinxcontrib-exceltable/


Module sphinxcontrib.exceltable is an extension for Sphinx, that adds support 
for including spreadsheets, or part of them, into Sphinx document.
