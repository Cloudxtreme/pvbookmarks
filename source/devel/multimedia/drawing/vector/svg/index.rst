===============================
SVG (Scalable Vector Graphics)
===============================

.. seealso::

   - http://fr.wikipedia.org/wiki/SVG
   - http://en.wikipedia.org/wiki/Scalable_Vector_Graphics

Le Scalable Vector Graphics (en français « graphique vectoriel adaptable »), 
ou SVG, est un format de données conçu pour décrire des ensembles de 
graphiques vectoriels et basé sur XML. 

Ce format inspiré directement du VML et du PGML est spécifié par le 
World Wide Web Consortium.
