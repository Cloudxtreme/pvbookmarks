Introduction
=============
   
Le logiciel libre Apache HTTP Server (Apache) est un serveur HTTP créé et maintenu 
au sein de la fondation Apache. 

C'est le serveur HTTP le plus populaire du World Wide Web. 

Il est distribué selon les termes de la licence Apache.
