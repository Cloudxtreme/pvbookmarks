new asynchronous features
-------------------------

In the .NET Framework 4.5 Developer Preview, new asynchronous features were
added to the C# and Visual Basic languages. These features add a task-based
model for performing asynchronous operations.

To use this new model, use the asynchronous methods in the I/O classes.
