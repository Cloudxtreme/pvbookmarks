========================
Java Virtual Machine
========================

.. seealso::

   - http://en.wikipedia.org/wiki/Java_Virtual_Machine
   - http://fr.wikipedia.org/wiki/Machine_virtuelle_Java


A Java virtual machine (JVM) is a virtual machine capable of executing Java
bytecode.

It is the code execution component of the Java software platform.

Sun Microsystems stated that there are over 4.5 billion JVM-enabled devices.
