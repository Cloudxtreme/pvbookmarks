Overview
=========

Inno Script Studio is a new intuitive graphical interface for generating and
compiling scripts for the award winning Inno Setup compiler from Jordan Russell.

Inno Script Studio provides help to simplify the creation of Microsoft Windows
installations by allowing the generation of Inno Setup scripts by use of the
intuitive interface to reduce the need to manually edit the Inno Setup scripts.

If you’ve created a freeware application, or even if it’s a large and complex
application, Inno Script Studio will help you create the perfect Windows
installation that fits both you and your customers’ requirements.
