=======================
Fuzzing Test
=======================

.. seealso:: https://secure.wikimedia.org/wikipedia/en/wiki/Fuzz_testing


Fuzz testing or fuzzing is a software testing technique that provides invalid,
unexpected, or random data to the inputs of a program.

If the program fails (for example, by crashing or failing built-in code assertions),
the defects can be noted
