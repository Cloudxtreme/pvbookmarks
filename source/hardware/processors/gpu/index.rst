Graphics Processing Unit
========================


A graphics processing unit (GPU), also occasionally called visual processing unit
(VPU), is a specialized electronic circuit designed to rapidly manipulate and
alter memory to accelerate the building of images in a frame buffer intended for
output to a display.

A video card is also a computer to itself.
